## Filenames
    While using the asset pipeline use hypens (-) to separate multi-word filenames and keep all words lower case.
    ```
    // bad
    MultiWordFilename.js

    // bad
    multiwordfilename.js

    // bad
    mulitWordFilename.js
    
    // bad
    multi_word_filename.js

    // good
    multi-word-filename.js
    ```
